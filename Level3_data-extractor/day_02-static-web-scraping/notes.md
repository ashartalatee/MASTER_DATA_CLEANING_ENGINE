# Day 02 – Static Web Scraping

## Apa yang saya pelajari
- Website punya struktur HTML
- Data bisa diambil via tag & class
- Tidak semua halaman bisa langsung diambil

## Struktur penting
- article.product_pod → container data
- h3 > a[title] → judul
- p.price_color → harga

## Insight
Scraping bukan tentang cepat,
tapi tentang memahami struktur HTML.
