# Day 6: Auto File Detection Engine

## Tujuan
- Mendeteksi file otomatis: CSV, XLS, XLSX, TSV
- Load file tanpa edit script
- Integrasi pipeline Day 1-5
- Modular, reusable, siap GitHub

## Tips
- Taruh semua dataset di folder data_input/
- Pipeline akan otomatis clean dan export ke data_output/
- Format file selain CSV/XLS/XLSX/TSV akan di-skip
