# Day 3: Numeric Cleaner Engine

## Tujuan
Convert semua kolom numeric ke tipe float/int, handle simbol & format angka.

## Kenapa penting
- Data analisis & AI butuh angka valid
- Format angka Excel/CSV bisa berbeda-beda (12,000 / Rp 12.000 / $12.000)
- Menjadi pondasi untuk feature engineering (Level 8)

## Tips
- Selalu test di dataset contoh dengan berbagai format angka
- Bisa pilih kolom spesifik jika tidak semua kolom numeric
