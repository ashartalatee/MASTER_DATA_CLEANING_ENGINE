# Column Cleaner Engine - Level 2 Day 1

## Deskripsi
Project ini membuat mesin otomatis untuk membersihkan nama kolom dataset CSV. 
Mendukung pipeline data automation di Level 2.

## Struktur Folder
- `data_input/` : tempat file mentah
- `data_output/` : tempat file hasil clean
- `cleaning_functions.py` : fungsi reusable
- `main.py` : run column cleaner

## Cara Pakai
1. Taruh file CSV di `data_input/`
2. Jalankan `python main.py`
3. File bersih otomatis tersimpan di `data_output/`
