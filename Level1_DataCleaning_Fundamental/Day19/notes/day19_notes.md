# Day 19 – Notes: Membuat Laporan Hasil Cleaning

Tujuan Day 19:
- Melatih skill komunikasi data → ini sangat penting di dunia kerja.
- Membuat laporan BEFORE → PROCESS → AFTER.
- Memisahkan data raw, processed, dan laporan.
- Merancang struktur proyek seperti professional data specialist.

Kenapa perlu laporan?
- Client tidak peduli Python, mereka peduli HASIL.
- Manager tidak mau baca kode, mereka mau ringkasan.
- Laporan membantu memastikan pekerjaan bisa di-audit.

Mindset hari ini:
"Data cleaning bukan hanya bersih, tapi bisa dijelaskan secara bisnis."

Yang harus ingat:
- Selalu simpan BEFORE dan AFTER.
- Jangan ubah data di folder raw/.
- Laporan harus bisa dibaca orang non-teknis.
- Script harus bisa digunakan ulang.

Catatan untuk masa depan:
Day 19 ini fondasi untuk Day 20 (mini client project).
Gunakan template laporan ini untuk semua proyek real.
