# Day 6 — Struktur Kolom & Tipe Data

## Tujuan Hari Ini
Memahami struktur dataset: kolom, tipe data, dan potensi error sebelum melakukan cleaning.

## Yang Saya Lakukan
- Memuat dataset mentah
- Melihat `df.info()`, `df.head()`, `df.describe()`
- Mencatat tipe data yang salah
- Mengidentifikasi kolom yang bermasalah
- Membuat file dokumentasi `column_structure.txt`

## Masalah yang Saya Temukan
Tuliskan poin singkat:
- Kolom price terbaca string
- Tanggal tidak konsisten
- Gender tidak standar
- Phone number kacau
- Ada kolom kosong

## Insight Hari Ini
Saya menyadari:
- Kesalahan tipe data adalah sumber error terbesar
- Data tidak bisa dianalisa sebelum tipe data benar
- Struktur kolom adalah pondasi cleaning & automation
